# ABC-MART 사이트

개인 공부 프로젝트로 React 사용경험 및 숙련도 향상을 위해 진행한 신발 쇼핑몰 사이트 입니다.

* Redux Toolkit을 이용한 상태관리
* Local Storage를 사용한 최근 본 상품 기능
* Bootstrap 사용
* lazy import하여 로딩속도 향상, 성능개선

🖥️ [사이트 보러가기](https://songyunjeong.github.io/abcmart_site)
